# ⚡ GigSpark

A mobile-first onboarding app for matching young job seekers with local gigs.

## Features
- 10-step guided profile builder
- Role selection (job seeker or hiring)
- Interest picker with custom entries
- Availability scheduling
- Travel distance preferences
- Experience builder (optional)
- Contact info collection
- Full review screen before submission

## Tech Stack
- React 18
- Vite
- Pure CSS (no external UI libraries)

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
